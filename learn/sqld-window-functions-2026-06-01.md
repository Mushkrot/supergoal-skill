# SQLD window functions — 2026-06-01

## Question
SQLD 현재 학습 주제로 윈도우 함수 이해, 예제, 실제 업무 활용을 원함. 난이도는 저장 난이도보다 한 단계 낮춤.

## Bridge
게임 랭킹표처럼 원본 참가자 목록은 그대로 두고, 각 줄 옆에 순위나 이전 점수 같은 계산 결과를 붙이는 방식으로 설명한다.

## Key terms
- **윈도우 함수** — 행을 없애지 않고, 각 행 옆에 계산 값을 붙이는 함수.
- **OVER** — 어디까지 보고 계산할지 알려주는 표시.
- **PARTITION BY** — 계산할 묶음을 나누는 기준.
- **ORDER BY** — 묶음 안에서 줄 세우는 기준.
