Implemented `deepMerge` improvements in `src/merge.mjs`.

Changed behavior:
- Recursively merges nested plain objects.
- Preserves existing nested sibling keys.
- Creates missing nested target branches without sharing source object references.
- Replaces arrays, `null`, primitives, dates, and other non-plain objects.
